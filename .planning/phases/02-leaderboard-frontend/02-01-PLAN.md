  <action>
Create frontend/src/pages/LeaderboardPage.js with the following features:

**State management:**
- `entries` - array of leaderboard entries
- `loading` - boolean for loading state
- `error` - error message or null
- `selectedGame` - current game filter (empty = all)
- `selectedMetric` - current sort metric ('wpm' | 'accuracy' | 'combined')
- `bestRecent` - 'best' | 'recent' toggle
- `currentUser` - decoded username from JWT for highlighting

**useEffect on mount:**
1. Initialize auth token: `setAuthToken(getAuthToken())`
2. Decode JWT to get username (for highlighting): Use `jwt-decode` package, extract `sub` or `username` claim
3. Fetch global leaderboard: `GET /api/leaderboards/global?metric=${selectedMetric}`
4. Set loading=false

**Game filter dropdown:**
- MUI Select with MenuItem for each game type: '', 'TypingTest', 'FallingTypingTest', 'GalaxyGame', 'GridGame', 'Bookworm', 'CrosswordGame'
- On change: set selectedGame, fetch from `/api/leaderboards/game/${gameId}?metric=${selectedMetric}`

**Metric toggle (segmented button group):**
- MUI ToggleButtonGroup with 3 options: WPM, Accuracy, Combined
- Position at top, separate section from filters
- Label: "Sort by: [Metric]" that updates dynamically
- Tooltip on Combined button: "Formula: WPM × (Accuracy/100) × 1.5 if accuracy > 95%"
- Debounce 300ms before API call (clearTimeout/setTimeout pattern)

**Best/Recent toggle:**
- MUI Switch with labels: "All-time" | "Recent"
- Position above table
- Show count: "Recent (10 games)" when recent selected
- State persisted to localStorage key 'leaderboard_best_recent'
- Debounce 300ms before API call

**Leaderboard table:**
- MUI TableContainer with Paper shadow
- Columns: Rank, Username, WPM, Accuracy, Combined Score, Game, Date
- Medal emoji for top 3: 🥇🥈🥉
- Current user row: highlighted with background color + border (check username match)
- Full date format using toLocaleDateString('en-US', { month: 'short', day: 'numeric', year: 'numeric' })

**Loading state:**
- Centered CircularProgress spinner

**Empty state:**
- Alert with message: "No scores yet. Play a game to get on the board!"
- Encouraging tone

**Error state:**
- Alert with error message
- Auto-retry after 5 seconds (setTimeout in useEffect)
- Manual refresh button

**Guest user banner:**
- If not logged in (no getAuthToken()): show fixed top banner with Register/Login buttons
- Below banner: show leaderboard table (publicly accessible)
