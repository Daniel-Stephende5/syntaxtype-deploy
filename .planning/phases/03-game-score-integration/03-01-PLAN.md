  <name>task 2: Add updateLeaderboardIfBetter method to LeaderboardService</name>
  <files>src/main/java/com/syntaxtype/service/LeaderboardService.java</files>
  <action>
Add method: `updateLeaderboardIfBetter(String userId, Category category, Integer wpm, Integer accuracy, Integer score)`

Logic:
1. Find existing Leaderboard entry for user + category
2. Calculate combined score: wpm * (accuracy/100) * (accuracy > 95 ? 1.5 : 1)
3. For typing games (TYPING_TESTS, FALLING_WORDS): compare combined scores
4. For non-typing games: compare raw scores
5. If new > existing: update Leaderboard with new best
6. Return {success, isNewBest, rank}

Handle case where no existing entry (first score).
  </action>
